## Introduction
This package contains a few extensions that are useful for developers working with Tiptap.

## About Tiptap

**Tiptap** is a highly customizable and modular editor for Vue.js. It comes with a wide range of features like tables, code blocks, text alignment, and even a full-featured Markdown editor.

## Installation
```bash
npm install tiptap-dev-tool
```
